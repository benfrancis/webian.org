Home web server

Webian Home is a web server for your home. The initial focus is on creating a home page for your home which provides a web interface to all your connected devices using the Web of Things. The broader vision includes all kinds of use cases around smart home, home media and family organisation.

Use Cases

Smart Home

  • Heating
  • Lighting
  • Security
Home Media
  • Family photos
  • Music collection
  • Home videos
Family organisation
  • Family calendar
  • Family todo list
  • Family noticeboard

Source Code

Webian Home is currently at the prototyping stage. You can find the source code on GitHub.
